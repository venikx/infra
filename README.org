#+TITLE: Personal Infrastructure

Personal cloud infrastructure managed as code.

* What's Here
AWS infrastructure using CDK + TypeScript:
- Backup systems for home services
- Core networking and IAM foundation
- Reusable constructs and utilities

* Getting Started
Prerequisites:
- Nix with flakes enabled
- direnv

Setup:
#+begin_src shell
direnv allow # Loads Nix environment
#+end_src
